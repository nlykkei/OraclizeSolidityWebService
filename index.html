Oraclize Solidity WebService


Array Sorting (Plain): /sort/n1/n2/...

   Returns the sorted array as space-separated plaintext.


Array Sorting (Binary): /sortb/n1/n2/...

   Returns the sorted array encoded as 16-bit big-endian (BE) integers.


Sqrt: /sqrt/n

   Returns the integer square-root as plaintext.


Min: /min/n1/n2/...

   Returns the mininum over an array of integers encoded as 16-bit integer.


3SUM: /3sum/n1/n2/...

   Returns a triplet of indicies of elements summing to zero encoded as 48-bit string.


All-Pairs Shortest Path: /apsp/n1/n2/...

   Returns the shortest path between each pair of vertices encoded as 16-bit integers.


Shortest Path: /sp/k/W/n1/n2/...

   Returns a shortest path of length 'k' and weight ≤ 'W' encoded as list of 16-bit integers.
