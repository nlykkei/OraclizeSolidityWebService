Oraclize Solidity Web Service


@state


Array Sorting: /sort/n1/n2/...

   Returns the sorted array encoded as 16-bit big-endian (BE) integers.


Square Root: /sqrt/n

   Returns the integer square-root as ASCII.


Array Min: /min/n1/n2/...

   Returns the mininum over an array of integers encoded as 16-bit BE integer.


3SUM: /3sum/t/n1/n2/...

   Returns a triplet of indices of elements summing to t encoded as 48-bit string


All-Pairs Shortest Path: /apsp/n1/n2/...

   Returns the shortest path lengths between each pair of vertices encoded as 16-bit BE integers.


k-Path: /kp/k/W/n1/n2/...

   Returns a path of length k and weight ≤ W encoded as list of 16-bit BE integers.


k-Dominating Set: /kds/k/n1/n2/...

   Returns a dominating set of size ≤ k encoded as list of 16-bit BE integers.


Echo: /echo/val

   Echo response as ASCII.
